# Student Marks Analysis

## 📖 Project Overview

This project performs an Exploratory Data Analysis (EDA) on student academic performance using Python. The analysis focuses on understanding how study habits, attendance, previous failures, parental education, internet access, and other factors influence students' final grades. The project uses data visualization and statistical analysis to uncover meaningful insights.

---

## 🎯 Objectives

- Analyze student academic performance using real-world data.
- Identify factors affecting final grades.
- Perform data cleaning and preprocessing.
- Visualize relationships between different variables.
- Generate meaningful insights from student performance data.

## 📈 Key Insights

- Students who dedicate more time to studying generally achieve higher final grades.
- Previous academic failures have a negative impact on student performance.
- High absenteeism is associated with lower academic achievement.
- Parental education and internet access show a positive relationship with student performance.
- Visualizations help identify trends and patterns in academic results.
